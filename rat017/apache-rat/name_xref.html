                                                                           form of "Copyright 2008

                          editOverwrite =                                  to be written directly to                  
                          'text'/>                                         the source files (i.e.                     
                                                                           new files are not                          
                                                                           created). Only valid with                  
                                                                           --edit-license                             

--edit-license          <rat:report editLicense  <editLicense>           Add the default license      -- none --

                                                                           to force the modification                  
                                                                           of existing files use the                  
                                                                           --edit-overwrite option.
