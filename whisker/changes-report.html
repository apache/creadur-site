Type Changes By
Update Jdom to 2.0.6.1 in order to fix CVE CVE-2021-33813. Fixes WHISKER-20. pottlinger
Upgrade Apache commons-io to fix CVE-2021-29425. Fixes WHISKER-19. pottlinger
Upgrade Apache Commons Collections to v3.2.2 to circumvent CVE-2015-8103. Fixes WHISKER-15. pottlinger
Fixes error during site generation with Java8 and maven plugin version 3.3. Fixes WHISKER-13. dennisl
Change artifactId/goalPrefix of Maven plugin to comply with expected apache-whisker-maven-plugin. Fixes WHISKER-14. dennisl
Update webpage to not load fonts from GoogleFonts and include privacy link in main menu. Fixes WHISKER-22. pottlinger
